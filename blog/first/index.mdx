You can write your blogs here!

make sure to add the followings to the top of your index.mdx document

1. Create a folder in the blog directory [In this case , I have named it as first]
2. Inside the Folder, create a new file called index.mdx .
3. Make sure to use template give in template.js in the same blog directory
4. title: "title of your blog"
5. date : "date of publishing"
6. author: "Your name"
7. slug: "blog-title-without-space"
